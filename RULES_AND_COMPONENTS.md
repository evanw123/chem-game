# Reaction Reactor

Reaction Reactor is an endless chemistry review video game where players type chemical formulas before the short timer expires.

## How to play

1. Press **Start Endless Run** to begin.
2. Read the mission card. Each mission gives the name of a compound and a clue about what it is.
3. Type the correct chemical formula into the **Formula Entry** box.
4. Use normal chemical notation with element symbols and numbers, such as `H2O`, `NaCl`, or `CaCO3`.
5. Press **Launch Reaction** to submit your answer before the timer runs out.
6. A correct answer scores points and immediately moves you to the next mission.
7. A wrong answer counts as 1 mistake.
8. If the timer reaches zero, that also counts as 1 mistake and the game reveals the correct formula for that round.
9. You begin with 3 mistakes available and 3 catalyst hints.
10. The game is endless, so your goal is to survive as many rounds as possible before reaching 3 mistakes.

## Original game components

This game includes more than half original components, which matches the assignment requirement.

- **Mission Deck**: 30 original chemistry challenge cards with custom clues, shorter timers, and point values.
- **Formula Entry Console**: A custom typed-answer system with live preview.
- **Formatting Guide Panel**: A built-in helper panel showing formula examples and notation tips.
- **Mistake Counter**: A clear 3-mistake loss system.
- **High Score + Leaderboard**: A persistent browser-saved ranking of the best endless runs.
- **Catalyst Hints**: Limited-use hints that reveal structural clues about the formula.
- **Score + Streak System**: Original scoring rules that reward speed and multiple correct answers in a row.
- **Mission Log**: A running record of wins, losses, and lab events.

## Educational focus

The game reviews:

- chemical formulas
- compound names
- atom ratios in compounds
- common chemistry substances
- quick identification of ionic and covalent compounds
- memorization of acids, oxides, salts, and nitrates commonly used in class examples
